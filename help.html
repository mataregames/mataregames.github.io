WHY IS THE WEBSITE NOT DISPLAYING PROPERLY?/WHY ARE THE WEBSITE FUNCTIONS NOT WORKING?

  • Perhaps your browser does not support the website.Try using Mozilla Firefox or Google Chrome
  • If the problem perssists try using a laptop or desktop computer
  • Make sure JAVASCRIPT is enabled in your browser(You can confirm this in your Browser Settings)

WHY CAN'T I USE THE Fullscreen OPTION?

  • Perhaps your browser does not support the FEATURE.Try using Mozilla Firefox or Google Chrome
  • Make sure that you have given the webpage permission to go into fullscreen mode


In general, for the best experience:

  1. Use Mozilla Firefox / Google Chrome
  2. Allow FullScreen and enable Javacsript
  3. Use a laptop/desktop computer
  4. Make sure you are using a good videoconferencing platform
  5. Make sure you have a good internet connection

Thank you and we hope you will find these suggestions useful
